# Dynamic Economic Regulator - 部署指南

## 项目概述

动态经济调节网络 (Dynamic Economic Regulator Network) 是一个为EVE Frontier设计的基于供需自动调节的经济系统。该系统通过智能存储单元和智能星门实现资源的动态分配和价格调节，为EVE Frontier的宇宙带来自我平衡的经济生态。

## 系统架构

### 核心智能合约

1. **ResourceOracle** - 资源监控与数据收集
   - 监控各区域资源存量和流动情况
   - 计算供需比例指数
   - 触发价格调整事件

2. **PriceRegulator** - 动态定价算法
   - 实现动态定价算法
   - 支持多种定价模型（固定价格、AMM模式、需求弹性定价等）
   - 防止价格操纵的约束机制

3. **TreasuryPool** - 多区域金库管理
   - 多区域金库管理
   - 跨区域资金自动调配
   - 收益分配机制

4. **DynamicGate** - 智能星门扩展
   - 根据资源流动自动调整通行费
   - 实现拥堵定价机制
   - 提供优先通行通道（付费优先）

### 前端应用

1. **EconomicDashboard** - 经济状态监控面板
   - 实时资源地图可视化
   - 价格趋势图表
   - 经济指标展示

2. **TradeInterface** - 交易执行界面
   - 资源查询与比较
   - 最优路径推荐
   - 交易执行界面

3. **AdminPanel** - 系统管理界面
   - 参数调节界面
   - 系统监控仪表
   - 收益管理功能

## 部署步骤

### 前置条件

1. **开发环境**
   - Node.js >= 16.0
   - pnpm 或 npm
   - Sui CLI (已安装)
   - EVE Vault钱包 (已安装)

2. **测试网络**
   - Sui Testnet访问权限
   - 测试SUI代币
   - 测试LUX代币

### 第一步：部署智能合约

1. **编译合约**
   ```bash
   cd move-contracts
   sui move build
   ```

2. **发布合约**
   ```bash
   sui client publish
   ```

3. **记录重要地址**
   发布成功后，记录以下地址：
   - Package ID
   - ResourceOracle对象ID
   - PriceRegulator对象ID
   - TreasuryPool对象ID

4. **初始化系统**
   使用初始化脚本创建示例系统：
   ```bash
   # 在部署脚本中调用初始化函数
   # 这将创建示例区域、价格数据和金库
   ```

### 第二步：配置前端应用

1. **安装依赖**
   ```bash
   cd dapp
   npm install
   ```

2. **配置合约地址**
   在 `src/utils/index.ts` 中更新合约地址：
   ```typescript
   export const CONTRACT_ADDRESSES = {
     DYNAMIC_ECONOMIC_REGULATOR: '0x...', // 实际部署的包ID
     RESOURCE_ORACLE: '0x...', // 实际的ResourceOracle对象ID
     PRICE_REGULATOR: '0x...', // 实际的PriceRegulator对象ID
     TREASURY_POOL: '0x...', // 实际的TreasuryPool对象ID
   };
   ```

3. **运行开发服务器**
   ```bash
   npm run dev
   ```

### 第三步：系统初始化

1. **连接钱包**
   - 打开前端应用 (http://localhost:3000)
   - 连接EVE Vault钱包
   - 确保钱包中有足够的测试SUI

2. **创建示例数据**
   - 访问管理面板 (/admin)
   - 使用"Create Region"按钮创建示例区域
   - 配置每个区域的初始参数

3. **验证系统运行**
   - 检查仪表板是否显示数据
   - 验证交易功能是否正常
   - 确认管理功能可以正常使用

## 使用指南

### 经济状态监控

1. **访问仪表板**
   - 导航到根路径 (/)
   - 查看实时经济指标
   - 监控价格趋势和资源分布

2. **理解指标**
   - **Active Regions**: 当前活跃的区域数量
   - **Total Volume**: 24小时交易总量
   - **Total Revenue**: 系统总收入
   - **Avg Price Change**: 平均价格变动百分比

### 资源交易

1. **创建交易**
   - 导航到交易页面 (/trade)
   - 选择源区域和目标区域
   - 输入交易数量
   - 查看费用估算

2. **使用优先通行证**
   - 勾选"Use Priority Pass"选项
   - 享受20%费用折扣
   - 适合高频交易者

3. **执行交易**
   - 确认交易详情
   - 点击"Execute Trade"
   - 等待交易确认

### 系统管理

1. **区域管理**
   - 导航到管理面板 (/admin)
   - 创建、编辑或删除区域
   - 监控区域状态

2. **定价配置**
   - 调整基础价格
   - 选择定价模型
   - 监控价格变化

3. **金库管理**
   - 查看各区域金库余额
   - 执行跨区域资金调配
   - 配置收益分配

## 故障排除

### 常见问题

1. **合约部署失败**
   - 检查Sui CLI配置
   - 确保有足够的SUI余额支付Gas费
   - 验证合约代码语法

2. **前端无法连接钱包**
   - 确保EVE Vault钱包已正确安装
   - 检查网络连接
   - 刷新页面重试

3. **数据不显示**
   - 验证合约地址配置是否正确
   - 检查网络连接
   - 查看浏览器控制台错误信息

### 调试技巧

1. **查看交易详情**
   ```bash
   sui client object <object-id>
   ```

2. **查询事件**
   ```bash
   sui client event --type <event-type>
   ```

3. **检查合约状态**
   - 使用Sui Explorer查看合约状态
   - 监控事件和交易历史
